
# インストール

1. ディレクトリごと、Next.js プロジェクトの app/api の名前で配置
1. Next.js プロジェクトを起動
1. localhost:3000/api/v1 にアクセスして、`{ "message": "ok" }` のレスポンスが返ってくることを確認

[Next.js での ルーティング設定](https://nextjs.org/docs/app/building-your-application/routing)
